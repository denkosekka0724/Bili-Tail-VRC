# 权限说明

## activeTab

用于在用户点击扩展图标后读取当前活动标签页的 URL，判断是否为 B 站视频页、列表页里的视频或直播间，并取得 BV 号、av 号、分P参数或直播间号；也用于只在该次点击触发后向当前标签页显示短暂结果提示。

扩展不会在后台持续读取标签页，也不会读取浏览历史。

## clipboardWrite

用于在用户点击扩展图标时，把解析得到的 91biliplayer 包装播放地址、B 站临时直链、直播直链或用户自建后端生成的带弹幕 HLS 地址写入剪贴板。

## contextMenus

用于在扩展图标菜单中显示“切换小尾巴配置”，让用户打开配置页切换普通 / 弹幕模式、直链清晰度、后端地址和弹幕参数。扩展不会读取或记录用户在浏览器菜单中的其他操作。

## scripting

用于在用户点击扩展图标后，只向当前活动标签页注入一次极小的脚本，读取当前分P/cid 等页面上下文，完成剪贴板写入 fallback，并显示 2 秒结果提示。弹幕模式下，这个脚本也会向用户自己填写的后端地址创建和查询任务。

扩展不会持续注入脚本，不会记录页面内容或用户操作。

## storage

用于保存用户在扩展配置页中的偏好设置，包括普通 / 弹幕模式、清晰度偏好、后端地址、弹幕数量上限和单条长度上限。设置只用于本扩展功能，不会上传到作者服务器。

## Host permission: `https://api.bilibili.com/*`

用于请求 Bilibili 官方 API，获取视频信息、分P cid 和播放地址。

## Host permission: `https://api.live.bilibili.com/*`

用于请求 Bilibili 直播官方 API，解析直播间真实 room id、开播状态和直播播放地址。

扩展不包含远程代码，不使用外部脚本。弹幕模式访问的后端地址由用户在配置页自行填写。

## Single Purpose

扩展的唯一用途是：把当前 B 站视频页、列表页里的视频或直播间解析成可复制的 VRChat/VRC 播放链接；在用户启用弹幕模式时，将普通 B 站视频交给用户自建后端生成带弹幕 HLS 地址。
