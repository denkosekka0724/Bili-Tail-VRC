Bili小尾巴 VRC解析 隐私政策

更新时间：2026-06-12

Bili小尾巴 VRC解析 是一个用于把当前 B 站视频页、列表页里的视频或直播间解析成 VRChat/VRC 播放链接的 Chrome 扩展。

我们收集什么

本扩展不收集、出售、共享或上传用户个人信息。

本扩展不会保存：

  • Bilibili 账号信息
  • Cookie 内容
  • 密码或 Token
  • 浏览历史记录
  • 生成的链接
  • 用户输入内容

本扩展会使用 Chrome 扩展存储保存用户偏好设置，包括：

  • 清晰度偏好

这些设置只保存在 Chrome 扩展存储中，用于扩展自身功能，不包含个人身份信息，也不会发送给作者服务器。

扩展如何工作

当用户点击扩展图标时，扩展会读取当前活动标签页的 URL，用于判断是否为 B 站视频页、列表页里的视频或直播间。对于分P视频，扩展会在本地读取当前页面的分P/cid 上下文，确保复制当前分P对应的地址。

如果当前页面是支持的视频页，扩展会向 Bilibili 官方 API 请求视频基础信息和播放信息，用于生成并复制：

  • 普通视频单击：91biliplayer 包装播放地址
  • 普通视频双击、分P视频或列表页视频：B 站临时直链
  • 直播间：直播临时直链

这些请求只用于完成用户主动触发的解析操作。扩展不会把数据发送到作者服务器，也没有作者服务器。

Cookie 说明

扩展请求 Bilibili API 时使用不携带凭据的请求方式。扩展不会读取、显示、保存、转发或主动发送用户 Cookie。

第三方服务

普通视频单击时，扩展生成的 VRChat 播放包装链接会使用：

https://biliplayer.91vrchat.com/player/?url=

该链接用于让用户在 VRChat/VRC 播放器中尝试播放 B 站页面链接。该 91biliplayer 第三方服务不是本扩展作者提供，其可用性和隐私实践不由本扩展控制。
