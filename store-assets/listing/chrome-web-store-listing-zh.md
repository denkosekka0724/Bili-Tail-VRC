# Chrome Web Store Listing

## Name

Bili小尾巴 VRC解析

## Short Description

摇摇小尾巴，把B站视频轻轻变成VRC播放链接，一键复制带去世界里用。

## Detailed Description

Chrome Web Store:
https://chromewebstore.google.com/detail/naeaecjabhcjeagenojhaiflljegmaca?utm_source=item-share-cb

Bili小尾巴 VRC解析是一个小巧的 Chrome 扩展，帮你在打开 B 站视频页、列表页里的视频或直播间时，一键复制可以带去 VRChat/VRC 世界里尝试使用的播放链接。

适合这些场景：

- 在 VRChat 世界里快速尝试播放 B 站视频
- 普通视频单击复制 91biliplayer 包装播放地址
- 普通视频双击复制 B 站临时直链
- 分P视频、列表/稍后再看页面里的视频自动复制当前视频直链
- B 站直播间自动尝试复制直播直链
- 可在扩展图标菜单中点击“切换小尾巴配置”，选择直链清晰度偏好
- 首次安装会自动打开本地可点击使用指南，用仿 Chrome 页面演示如何点击右上角小尾巴拿到链接

使用方式：

1. 打开一个 B 站普通视频页、分P视频页、带 `bvid`/`aid`/`oid` 的列表页，或 B 站直播间。
2. 点击浏览器右上角的扩展图标。
3. 页面顶部中上方出现成功提示后，对应地址已经复制到剪贴板。
4. 普通视频需要临时直链时，快速双击扩展图标即可。
5. 如需调整直链清晰度，在扩展图标菜单中点击“切换小尾巴配置”。

说明：

- 这是非官方工具，不隶属于 Bilibili 或 VRChat。
- 普通视频单击会生成 `biliplayer.91vrchat.com` 的第三方包装播放地址；该服务不是本扩展作者提供。
- B 站临时直链通常有时效，可能会失效。
- VRChat 播放结果取决于播放器、网络环境、第三方解析服务和视频权限。
- 请只播放你有权使用的内容。
