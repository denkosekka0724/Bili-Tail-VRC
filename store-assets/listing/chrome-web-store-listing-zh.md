# Chrome Web Store Listing

## Name

Bili小尾巴 VRC弹幕版

## Short Description

把B站视频变成VRC可用链接，也能连接自己的后端生成带弹幕HLS。

## Detailed Description

这是 Bili小尾巴 VRC 的弹幕实验版。普通模式和正式版保持同款操作：普通视频单击复制 91biliplayer 播放地址，普通视频双击复制 B 站临时直链，分P、列表页视频和直播间会自动走直链。弹幕模式额外支持连接你自己填写的后端，把当前 B 站普通视频生成带弹幕的 HLS 地址，再复制给 VRChat/VRC 播放器测试。

适合这些场景：

- 在 VRChat 世界里快速尝试播放 B 站视频
- 普通模式单击复制 91biliplayer 包装播放地址
- 普通模式双击复制 B 站临时直链
- 分P视频、列表/稍后再看页面里的视频自动复制当前视频直链
- B 站直播间自动尝试复制直播直链
- 弹幕模式连接自建后端，生成带弹幕 HLS 地址
- 可在扩展图标菜单中点击“切换小尾巴配置”，选择普通 / 弹幕模式、直链清晰度、后端地址和弹幕数量
- 首次安装会自动打开本地可点击使用指南，用仿 Chrome 页面演示如何点击右上角小尾巴拿到链接

使用方式：

1. 打开一个 B 站普通视频页、分P视频页、带 `bvid`/`aid`/`oid` 的列表页，或 B 站直播间。
2. 普通模式下点击浏览器右上角扩展图标，页面顶部出现成功提示后，对应地址已经复制到剪贴板。
3. 普通视频需要临时直链时，快速双击扩展图标即可。
4. 如需带弹幕版本，先在扩展图标菜单中打开“切换小尾巴配置”，切到带弹幕 HLS，并填写自己的后端地址。
5. 回到普通 BV/av 视频页后单击扩展图标，等待后端生成 HLS 地址并自动复制。

说明：

- 这是非官方实验工具，不隶属于 Bilibili、VRChat 或 91biliplayer。
- 普通视频单击会生成 `biliplayer.91vrchat.com` 的第三方包装播放地址；该服务不是本扩展作者提供。
- 弹幕模式需要用户自己部署后端和 ffmpeg；扩展作者不提供弹幕后端服务。
- B 站临时直链通常有时效，可能会失效。
- VRChat 播放结果取决于播放器、网络环境、第三方解析服务、自建后端和视频权限。
- 请只播放、转码和展示你有权使用的内容。
